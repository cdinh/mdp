byte FarseerPhysics.Common.TextureConverter.AlphaTolerance
getset

Alpha (coverage) tolerance.

Default is 20: Every pixel with a coverage value equal or greater to 20 will be counts as solid.

bool FarseerPhysics.Common.TextureConverter.HoleDetection
getset

Will detect texture 'holes' if set to true.

Slows down the detection. Default is false.

float FarseerPhysics.Common.TextureConverter.HullTolerance
getset

Default is 1.5f.

bool FarseerPhysics.Common.TextureConverter.MultipartDetection
getset

Will detect texture multiple 'solid' isles if set to true.

Slows down the detection. Default is false.

bool FarseerPhysics.Common.TextureConverter.PixelOffsetOptimization

Will optimize the vertex positions along the interpolated normal between two edges about a half pixel (post processing).

Default is false.
