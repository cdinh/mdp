Member Data Documentation

float FarseerPhysics.Controllers.BuoyancyController.AngularDragCoefficient

Controls the rotational drag that the fluid exerts on the bodies within it.

Use higher values will simulate thick fluid, like honey, lower values to simulate water-like fluids.

float FarseerPhysics.Controllers.BuoyancyController.Density

Density of the fluid.

Higher values will make things more buoyant, lower values will cause things to sink.

float FarseerPhysics.Controllers.BuoyancyController.LinearDragCoefficient

Controls the linear drag that the fluid exerts on the bodies within it.

Use higher values will simulate thick fluid, like honey, lower values to simulate water-like fluids.

FVector2 FarseerPhysics.Controllers.BuoyancyController.Velocity
