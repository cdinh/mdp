Detailed Description

The class manages contact between two shapes.

A contact exists for each overlapping AABB in the broad-phase (except if filtered). Therefore a contact object may exist that has no contact points.

bool FarseerPhysics.Dynamics.Contacts.Contact.Enabled

Enable/disable this contact.

This can be used inside the pre-solve contact listener. The contact is only disabled for the current time step (or sub-step in continuous collisions). NOTE: If you are setting Enabled to a constant true or false, use the explicit Enable() or Disable() functions instead to save the CPU from doing a branch operation.
